# 秋城生活指南 — 项目上下文

## 产品概述

**产品名称**：秋城生活指南  
**形态**：微信小程序  
**定位**：昭通全域深度旅行内容平台，聚合官方精品内容 + 用户真实攻略，服务短途观光、深度游、避暑旅居三类游客。

昭通市当前定位为"秋城"（避暑胜地），大量四川、重庆游客夏季来此旅居避暑（17-22°C），这是核心差异化场景。

---

## 目标用户

| 用户类型 | 核心诉求 | 优先级 |
|---------|---------|--------|
| 川渝避暑旅居客 | 选地、气候对比、租房参考、生活配套 | P0 核心 |
| 自驾游客 | 跨县路线、打卡点、营地、加油/充电桩 | P1 |
| 本地居民 | 探索周边区县、精品玩法 | P2 |

---

## 内容架构（三层）

```
内容来源
├── 官方精品层（编辑维护）
│   ├── 每个区县 1-2 篇精品攻略（共 22 篇，MVP 先做昭阳+巧家）
│   ├── 精品自驾路线（编辑策划）
│   └── 旅居生活配套指南
│
├── 用户攻略层（UGC，小程序内发布）
│   ├── 用户发布结构化图文游记
│   ├── 提交后 Claude API 自动审核（claude-haiku-4-5）
│   └── 通过后展示，拒绝时给用户反馈原因
│
└── 精选整理层（编辑从外部平台整理）
    ├── 来源：小红书、马蜂窝、知乎等
    ├── 摘录改写 + 底部注明来源
    └── 二期实施
```

**内容节奏**：每区县先出 1-2 篇官方精品攻略打底 → 开放用户投稿 → 外部攻略整理补充长尾

## 功能模块（完整）

### Tab 导航（4个）
- 首页 / 区县 / 旅居 / 自驾

### 首页
- 全局搜索
- 今日推荐攻略（官方+UGC混合）
- 快捷入口：想避暑 / 想自驾 / 想深度游 / 了解昭通
- 当季推荐（根据月份动态）
- 昭通主城实时天气（和风天气 API）

### 区县探索
- 11 个区县列表/地图入口
- 区县详情页：实时天气 + 基础信息 + 官方精品攻略 + 景点/美食/住宿 + 生活配套 + 用户游记聚合

### 攻略广场（UGC核心）
- 游记列表（全昭通 / 按区县筛选）
- 按类型筛选：景点 / 美食 / 旅居 / 自驾 / 小众秘境 / 住宿
- 按季节筛选：春夏秋冬
- 游记详情（图文）
- 收藏、点赞

### 发布游记（UGC发布）
- 多图上传（最多9张，云存储）
- 正文（≥100字）
- **结构化标签**：类型 + 季节 + 区县 + 关联景点（关键：用于在景点详情页聚合真实体验）
- 提交 → Claude API 自动审核 → 结果通知用户

### 行程定制器（新增）
- 三步流程：选区县 → 选天数（1-7天）+ 主题（6种）→ 自动生成行程预览
- 基于区县景点数据智能排列
- 6种主题：深度游 / 避暑旅居 / 自驾探索 / 亲子游 / 摄影采风 / 文化历史
- 行程可保存到"我的"或分享给朋友

### 旅居专区
- 避暑选地对比（11区县气候/海拔/旅居评分 + 实时天气）
- 长住攻略（交通/租房/生活）
- 租房参考价（各区县月租区间，参考值非实时）
- 旅居者真实故事（UGC精选）

### 自驾路线
- 路线列表（官方策划 + 用户分享）
- 路线详情：途经打卡点 / 营地 / 加油站 / 充电桩
- 新能源车专属提示

### 搜索
- 全局：景点名 / 区县 / 游记关键词 / 美食

### 我的
- 我的收藏 / 我的游记（含审核状态）/ 我的足迹 / 我的行程
